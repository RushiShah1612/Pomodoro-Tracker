## ‚ú® Features

-   **Timer Modes**: Switch effortlessly between Focus, Short Break, and Long Break modes.
-   **Customizable Durations**: Set your own preferred times for each mode in the settings.
-   **Dark Mode**: Fully supported dark theme that respects your system preference or can be toggled manually.
-   **Progress Tracking**: Dashboard to visualize your focus sessions and total focus time.
-   **Engaging Animations**: Celebration confetti on completing focus sessions to keep you motivated.
-   **Smart Notifications**: Toast notifications to alert you when a session ends, even if you're looking away.
-   **Data Persistence**: Your settings and progress are saved automatically to your browser's local storage.
-   **Responsive Design**: Works beautifully on desktop, tablet, and mobile devices.

## üöÄ Getting Started

Follow these steps to get the project up and running on your local machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) (v16 or higher) installed.

### Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/yourusername/pomodoro-tracker.git
    cd pomodoro-tracker
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Start the development server:**

    ```bash
    npm run dev
    ```

    The application will be available at `http://localhost:5173`.

## üìñ Usage

1.  **Start a Session**: Select "Focus" and click play to start the timer.
2.  **Take a Break**: When the timer ends, you'll be notified. Switch to "Short Break" or "Long Break".
3.  **Customize**: Click the **Settings (Gear Icon)** to adjust timer durations.
4.  **Toggle Theme**: Use the **Moon/Sun Icon** in the top right to switch between dark and light modes.
5.  **Track Progress**: Check the dashboard to see your daily stats.
